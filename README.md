# My Link Hub

A simple, single-page "link hub" site (similar to Linktree) built with plain HTML, CSS, and JavaScript.

## What It Does

Displays a personal profile with a collection of link buttons that point to your social profiles, portfolio, blog, and other pages. Features a dark theme, hover animations, and a staggered fade-in effect.

## Project Structure

```
my-link-hub/
├── index.html    ← Page structure and content
├── style.css     ← All visual styling (dark theme, animations)
├── script.js     ← Lightweight click interactions
├── README.md     ← You're reading it
└── .gitignore    ← Files Git should ignore
```

## How to Use

1. Open `index.html` in any browser — no build step or server needed.
2. Edit the `<a>` tags in `index.html` to point to your real URLs.
3. Customize colors in `style.css` (look for the hex values and gradients).

## Deployment

This site is designed to be deployed on **Vercel** via a connected GitHub repo. Every push to `main` triggers an automatic redeploy.
